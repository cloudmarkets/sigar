SIGAR to bundler gem
====================

This logs the conversion steps necessary to create a bundler gem from the latest sigar sources.

* Update Rakefile to use Gem::PackageTask: https://github.com/jackthecoder/sigar/commit/1b9dcdae8dd36bb82f7da3ed9ecb87dcfd9f9568

* Create new gem structure by bundle gem custom-sigar

* Split original Rakefile:
  - properties in gemspec
  - tasks in target Rakefile
  
* Adjust directory structure according to http://guides.rubygems.org/gems-with-extensions/
  - include/, src/ ... sigar/* => ext/cloudscale/sigar
  - bindings/ruby/* => lib/cloudmarkets/sigar
  - extconf.rb => ext/cloudmarkets/sigar
